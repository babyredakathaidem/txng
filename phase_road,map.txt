Prompt 2.2 — TransformationEventController

CONTEXT:

- Transformation event: biến đổi trạng thái, TIÊU THỤ lô input, SINH lô output mới

NHIỆM VỤ:
Viết app/Http/Controllers/TransformationEventController.php

Methods:
1. store(Request $request): RedirectResponse
   Validate:
   - cte_code: required|string
   - event_time: required|date
   - who_name: required|string|max:255
   - trace_location_id: nullable|exists:trace_locations,id
   - kde_data: nullable|array
   - note: nullable|string
   - input_batches: required|array|min:1
   - input_batches.*.id: exists:batches,id
   - input_batches.*.quantity: nullable|numeric|min:0.001
   - input_batches.*.unit: nullable|string
   - output_product_id: required|exists:products,id — sản phẩm của lô output
   - output_quantity: required|numeric|min:0.001
   - output_unit: required|string|max:50
   - output_batch_type: required|in:raw_material,wip,finished
   - output_production_date: nullable|date
   - output_expiry_date: nullable|date

   Logic trong DB::transaction():
   a. Validate tất cả input batches thuộc tenant + status='active'
   b. Tạo TraceEvent với event_category='transformation'
   c. Generate event_code
   d. Gắn input batches vào event_input_batches
   e. Update input batches: status='consumed', current_quantity=0
   f. Auto-sinh Batch output mới:
      - enterprise_id = tenantId
      - origin_event_id = event vừa tạo
      - code = generateBatchCode() với prefix theo category
      - batch_type = output_batch_type
      - product_id = output_product_id
      - gtin_cached = QrCodeService::resolveGtin()
      - gs1_128_label = GS1Service::buildGS1_128()
      - status = 'active'
   g. Gắn output batch vào event_output_batches
   h. Tạo BatchLineage records (giữ compatibility với code cũ):
      - transformation_type = xác định từ cte_code
      - input_batch_id, output_batch_id
   i. Gọi QrCodeService::ensureForBatch() cho output batch

   Return: redirect đến trang batch output vừa tạo
   Flash message: "Đã tạo sự kiện + lô [code]. Vui lòng điền đủ dữ liệu và publish."

2. show(Request $request, TraceEvent $event): Response
   Trả về Inertia page TransformationEvents/Show với đầy đủ:
   - event data
   - inputBatches với events của từng batch
   - outputBatches
   - location data

LƯU Ý QUAN TRỌNG:
- Batch input consumed KHÔNG thể undo sau khi event published
- Khi event còn draft → cho phép hủy (rollback consumed status)
- destroy() phải: restore input batches status='active', xóa output batch, xóa lineage

🟢 PHASE 4 — GS1 / TT02 Compliance
Prompt 4.1 — GS1 Enforcement
Bạn là Laravel developer chuyên sâu.

CONTEXT:
Thông tư 02/2024/TT-BKHCN + TCVN 13274:2020
Các AI bắt buộc trong hệ thống:
  AI(01)  — GTIN-14 sản phẩm
  AI(10)  — Mã lô
  AI(11)  — Ngày sản xuất YYMMDD
  AI(17)  — Hạn sử dụng YYMMDD
  AI(251) — Mã tham chiếu nguồn gốc (event_code)
  AI(400) — Số chứng từ (hợp đồng, phiếu kiểm)
  AI(414) — GLN địa điểm vật lý
  AI(416) — GLN địa điểm sản xuất/vùng trồng
  AI(417) — GLN doanh nghiệp
  AI(00)  — SSCC vận chuyển

NHIỆM VỤ:
Mở rộng app/Services/GS1Service.php — KHÔNG xóa method cũ

Thêm methods:

1. buildFullLabel(Batch $batch, ?TraceEvent $event = null): array
   Return tất cả AI codes cho 1 lô:
   {
     ai01: "(01)08938503XXXXXX",
     ai10: "(10)LOT-LUA-XYZ-202603-001",
     ai11: "(11)260315",  // production date
     ai17: "(17)270315",  // expiry date (nếu có)
     ai251: "(251)EVT-XYZ-HARVEST-202603-001",  // nếu có event
     ai416: "(416)8930000001234",  // nếu có location
     ai417: "(417)8930000001001",  // GLN enterprise
     gs1_128: "(01)...(10)...(11)...(17)...",  // full string
     digital_link: "https://domain/01/08938503.../10/LOT-..."
   }

2. buildSscc(int $enterpriseId, int $seq): string
   Format SSCC 18 số: (00)0{extension}{companyPrefix}{serial}{check}
   Extension digit = 0-9 (dùng 0 mặc định)
   Serial = seq padded

3. validateEventCode(string $code): bool
   Validate format: EVT-{2-10 chars}-{2-10 chars}-{6 digits}-{3 digits}

4. parseGs1String(string $gs1): array
   Parse chuỗi GS1-128 → array [ai => value]
   VD: "(01)08938503...(10)LOT-..." → ['01' => '08938503...', '10' => 'LOT-...']

Thêm validation rule:
app/Rules/ValidGln.php — validate GLN 13 số + check digit
app/Rules/ValidGtin.php — validate GTIN 8/12/13/14 + check digit
app/Rules/ValidEventCode.php — validate event code format
